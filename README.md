# Project Title
Head_Up_Notification_with_Action
# Description
Simple android head up notification app with action.
# Usages
### Build gradle
````
allprojects {
    repositories {
        maven{
            url "https://maven.google.com"
        }
        jcenter()
        google()
    }
}
````
### permisions
``````
<uses-permission android:name="android.permission.VIBRATE" />
``````
